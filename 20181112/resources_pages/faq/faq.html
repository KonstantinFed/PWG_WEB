FAQ

Questions and answers.

Q: In which type of games can be used PWG SDK?

A: Actually, it depends on game mechanics. PWG SDK is supposed to create worlds for Action RPG, top-down RPG, FPS, TPS, Roguelike games. That does not means that other types can't use PWG SDK. There is no restrictions for game type.

Q: What type of projects can use PWG SDK?

A: First of all, PWG SDK is developed for any C++ applications. Unreal Engine support will be also included in release. C++ is not only the way for using PWG SDK. If you don't need to re-create world during runtine and PWG SDK is needed only for help artist to create the world, you can just export models, textures and everything else from PWG Console.
